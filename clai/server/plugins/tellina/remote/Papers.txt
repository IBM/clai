Steps for setting up tellina using this version:-
a) uninstall any TensorFlow and tensorboard version. It would be good to delete the local install folder of tensorboard under the Python version. 
b) Make a fresh install using the following command:- pip install tensorflow==2.0.0-rc0
c) make clean
d) follow procedure in the README provided.
